arpwatch
===

監聽網路上ARP的記錄

## 補充說明

**arpwatch命令** 用來監聽網路上arp的記錄。

### 語法

```
arpwatch(選項)
```

### 選項

```
-d：啟動排錯模式；
-f<記錄檔案>：設定儲存ARP記錄的檔案，預設為/var/arpwatch/arp.dat；
-i<介面>：指定監聽ARP的介面，預設的介面為eth0；
-r<記錄檔案>：從指定的檔案中讀取ARP記錄，而不是從網路上監聽。
```
