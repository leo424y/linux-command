arptables
===

管理ARP包過濾規則表

## 補充說明

**arptables命令** 用來設定、維護和檢查Linux核心中的arp包過濾規則表。

### 語法

```
arptables(選項)
```

### 選項

```
-A：向規則鏈中追加規則；
-D：從指定的鏈中刪除規則；
-l：向規則鏈中插入一條新的規則；
-R：替換指定規則；
-P：設定規則鏈的預設策略；
-F：重新整理指定規則鏈，將其中的所有規則鏈刪除，但是不改變規則鏈的預設策略；
-Z：將規則鏈計數器清零；
-L：顯示規則鏈中的規則列表；
-X：刪除指定的空使用者自定義規則鏈；
-h：顯示指令幫助資訊；
-j：指定滿足規則的新增時的目標；
-s：指定要匹配ARP包的源ip地址；
-d：指定要匹配ARP包的目的IP地址。
```
